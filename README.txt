This is a window application.

HOW TO USE -- 
1. Run GaSchedule.exe
2. Goto File and select Find Configuration.
3. Select the file GaSchedule.cfg.
4. Again goto file and select Generate Schedule.
5. Wait till the Fitness value reaches 1.
6. Time Table is generated.

For Testing --
1. Install Visual Studio 2008 to open the project.
